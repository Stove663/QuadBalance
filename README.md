# QuadBalance

QuadBalance 是一个面向中国市场的四象限组合回测引擎，支持基于资产、策略与风控规则的组合构建、回测、压力测试和结果分析。

## 项目定位

- 面向中国资产市场的组合回测与验证
- 支持投资者画像、资产配置与适配性校验
- 提供策略扫描、指标计算、压力测试与报告输出能力
- 适合作为研究、策略验证和组合管理的实验平台

## 主要能力

- 资产池与工具目录管理
- 投资组合模板与配置参数管理
- 回测仿真与绩效指标计算
- 业绩归因、基准比较与敏感性分析
- 风险约束、适配性验证与压力测试
- 报告分节输出与结果汇总

## 环境要求

- Python 3.11 或更高版本

## 安装

推荐使用 `uv` 或其他现代 Python 包管理工具安装：

```bash
uv sync
```

如果你只想安装运行所需依赖，也可以使用：

```bash
pip install -e .
```

## 开发依赖

安装测试依赖：

```bash
pip install -e ".[dev]"
```

## 命令行入口

项目提供命令行入口 `quadbalance`。

查看可用命令：

```bash
quadbalance --help
```

## 常见工作流

1. 准备资产池与策略配置
2. 定义投资者画像、风险阈值和组合模板
3. 运行回测或参数扫描
4. 生成指标、压力测试与报告
5. 根据验证结果调整策略参数

## 目录说明

- `src/quadbalance/`：核心实现
- `tests/`：测试用例
- `openspec/`：需求、设计和任务说明
