## Why

Harry Browne 的永久投资组合（Permanent Portfolio）以四象限资产覆盖繁荣、通缩、通胀、衰退四种经济情景，但原版标的（美股、美债、黄金、美元现金）无法直接用于中国个人投资者。需要一套可在中国执行的买入策略——含机械规则、本土资产映射、以及用回测与压力测试验证后再锁定比例的流程——作为后续回测工具与实盘执行的共同契约。

## What Changes

- 定义中国版永久投资组合的四象限框架与默认目标比例（25/25/25/25，待回测后可调整）
- 固化机械执行规则：底仓一次性建仓、持续定投、年度再平衡（含触发阈值）
- 建立候选资产池：A 股宽基 + QDII 全球敞口（股票象限内 40% 全球，组合整体约 10%）、国债 ETF、黄金 ETF、场内货币基金
- 定义回测区间、基准对比、核心指标与压力情景的通过标准
- 明确策略锁定流程：回测与压力测试达标后方可最终确定比例与标的
- 本 change 仅产出策略规格与设计，回测引擎实现留待后续 change

## Capabilities

### New Capabilities

- `investment-strategy`: 四象限目标配置、底仓/定投/年度再平衡的机械规则与参数
- `asset-universe`: 中国可投资标的映射（主选与备选 ETF）、股票象限内 A 股/QDII 60/40 拆分
- `strategy-validation`: 回测方法、压力测试情景、验收指标与策略锁定门槛

### Modified Capabilities

（无——项目尚无既有 spec）

## Impact

- 新增 `openspec/specs/` 下三个 capability spec 文件
- 为后续 Python 回测模块（akshare 数据源、组合净值计算、再平衡模拟）提供需求基线
- 无现有代码或 API 变更；首个实现 change 将基于此策略规格构建回测引擎
